# 📚 Biblioteca da Vila

Sistema web desenvolvido para simular uma biblioteca comunitária, permitindo o cadastro, visualização, doação e reserva de livros de forma simples e intuitiva.

## 📖 Sobre o Projeto

A Biblioteca da Vila foi criada com o objetivo de incentivar o compartilhamento de livros e promover o acesso à leitura dentro de uma comunidade.

A aplicação permite que usuários cadastrem livros para doação, pesquisem obras disponíveis e realizem reservas através de uma interface moderna e responsiva.

Todo o gerenciamento de dados é realizado localmente utilizando o LocalStorage do navegador, dispensando a necessidade de um banco de dados ou servidor backend.

---

## ✨ Funcionalidades

### 📚 Gerenciamento de Livros

* Cadastro de novos livros.
* Edição de livros cadastrados.
* Exclusão de livros.
* Pesquisa de livros por título.
* Visualização de livros disponíveis.

### 🎁 Sistema de Doação

* Cadastro de livros para doação.
* Área exclusiva com os livros doados pelo usuário.
* Identificação automática do doador através de um ID único gerado localmente.

### 📦 Sistema de Reservas

* Adição de livros à lista de reserva.
* Contador dinâmico de reservas.
* Sidebar lateral para visualização das reservas.
* Cancelamento de reservas antes da confirmação.
* Confirmação de reserva.
* Alteração automática do status do livro para "Indisponível".

### 💬 Formulário de Contato

* Formulário para envio de mensagens.
* Feedback visual ao usuário após o envio.

### 🎨 Interface Moderna

* Hero section com imagem de destaque.
* Header responsivo.
* Sidebar animada.
* Modal para cadastro de livros.
* Accordion informativo na seção "Sobre".
* Scrollbar personalizada.
* Layout responsivo para dispositivos móveis.
